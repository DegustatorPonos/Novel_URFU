Евгений Викторович по сюжету принимает антидепрессанты, поэтому его речь в основном состоит из бормотания.
Можего прописать его вручную, но я написал для этого функцию.
Вызывается она так:
	$SayWGrumbling((str)[content],(int)[ammount])
где content - его фраза, а ammount - количество бормотания, ей предшествующее. 

Для изменения переменной персонажа есть функция
	$AttUpdate('<Character>', <ammount>)
Из-за особенностей развития языка персонажа ей надо давать именно в формате строки.
